# mcgl-slimes

Userscript для Tampermonkey, который отображает слайм-чайнки на карте проекта [mcgl.ru](https://minecraft-galaxy.ru/ru/). Сиды основных игровых планет уже встроены в скрипт, для КМ доступен функционал мануального ввода

## Установка

1. Установите [Tampermonkey](https://www.tampermonkey.net/)
2. Скачайте `index.user.js` из [последнего релиза](../../releases/latest)
3. Откройте панель Tampermonkey → Создать новый скрипт
4. Вставьте содержимое `index.user.js` и сохраните

На карте появится кнопка **Слизни**. Нажмите чтобы показать/скрыть слой слайм-чайнков. Кнопка **Ввести сид** позволяет указать сид вручную.

## Сборка

Требуется [Node.js](https://nodejs.org/) и [ldc2](https://github.com/ldc-developers/ldc/releases).

```bash
npm run build
```

Результат сборки в `dist/`.

## Сиды

Известные сиды миров хранятся в объекте `SEEDS` в файле `userscript.js`. Добавьте id мира и сид туда если он отсутствует.
